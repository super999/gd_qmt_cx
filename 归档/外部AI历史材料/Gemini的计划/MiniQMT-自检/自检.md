## 🛠️ MiniQMT 环境自检

要运行 MiniQMT，你需要确认以下三点：

1.  **QMT 客户端正在运行**（且登录了仿真或实盘账号）。
    
2.  **开启 API 服务**：通常在 QMT 顶部的 `策略交易` -> `外部 API` 或 `设置` 中勾选“允许第三方程序调用”。
    
3.  **库文件就绪**：检查 `D:\光大证券金阳光QMT实盘\bin.x64`（或类似 bin 目录）下是否有 `xtquant` 文件夹。
    

### 检查脚本 (Check\_MiniQMT.py)

你可以直接在你的 Python 环境里运行下面这段代码。如果能打印出你的“账号 ID”或“行情数据”，说明链路已通。

```python
import sys
import time

# 1. 动态引入 QMT 库路径（请根据你实际的 bin 路径调整）
# 这里的路径通常是 QMT 安装目录下的 xtquant 文件夹所在位置
sys.path.append(r'D:\、光大证券金阳光QMT实盘\\bin.x64\\Lib\\site-packages')

try:
    from xtquant import xtdata
    from xtquant.xttrader import XtQuantTrader, XtQuantTraderCallback
    print("✅ xtquant 库加载成功")
except ImportError:
    print("❌ 未找到 xtquant 库，请检查路径设置")
    sys.exit()

# 2. 测试行情连接 (无需登录交易账号即可测试)
def test_data():
    stock_code = '600519.SH' # 以贵州茅台为例
    xtdata.download_history_data(stock_code, period='1d', start_time='20260401')
    data = xtdata.get_market_data(stock_list=[stock_code], period='1d')
    if data:
        print(f"✅ 行情连接正常，{stock_code} 最新数据已获取")
    else:
        print("⚠️ 行情库已加载，但未获取到数据，请确认 QMT 客户端已启动且数据已下载")

if __name__ == "__main__":
    test_data()
```

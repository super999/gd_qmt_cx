# 归档目录说明

## 目的

本目录用于保存历史材料，而不是当前主线事实。

这里的文件可以帮助回答：

- 过去讨论过什么
- 某个判断最初是怎么来的
- 某轮实验或旧计划当时怎么写的

但它们默认不能直接覆盖当前结论。

## 目录分工

### `外部AI历史材料/`

保存来自外部 AI 的旧计划、旧判断、旧工作稿。

例如：

- `ChatGPT` 历史计划
- `Gemini` 历史计划

用途：

- 查历史思路
- 对照某些旧判断

不应用途：

- 不应直接当当前任务清单
- 不应直接当当前环境事实

### `历史产物/`

保存较早期的回测结果、历史产物、旧版本输出。

用途：

- 审计
- 回看历史阶段
- 对照项目演进

## 与当前主线的关系

当前主线文档仍在：

- `报告/项目管理/`
- `报告/环境与规范/`
- `报告/策略设计/`
- `报告/研究结论/当前主线/`

如果新会话只读归档而不读当前主线，很容易被旧材料带偏。
